

# Rate Repository App

A mobile application for rating GitHub repositories. Users can browse GitHub-style repositories, view individual repository details and reviews, and create new reviews when signed in.

The application is built with React Native and Expo, and it uses a GraphQL API as its backend. Created as part of Full Stack Open, part10
